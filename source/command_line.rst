Git Extensionsのコマンドライン
------------------------------

ほとんどの機能はコマンドラインから実行することもできます。
コマンドラインを使用する場合、 ``gitex.cmd`` をパスに追加することをお勧めします。
通常は、 ``C:\Program Files (x86)\GitExtensions`` フォルダに保存されています。
